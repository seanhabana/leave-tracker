Setup Instructions

  1. Create a new Google Sheet with these column headers in row 1:
    Name | Date From | Date To | Leave Type | Reason | Status
  2. Go to File → Share → Publish to web → publish as CSV
  3. Copy your Sheet ID from the URL: https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit
  4. Open index.html and replace YOUR_SHEET_ID_HERE with your Sheet ID
  5. Push to your GitHub repo and enable GitHub Pages (Settings → Pages → Deploy from branch → main)
  6. Share the GitHub Pages URL with your managers

Managers log absences directly in the Google Sheet. The calendar refreshes automatically.
